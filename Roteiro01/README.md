# 💻 Roteiro 1: Introdução à POO e Entrada de Dados

Neste pacote estão os códigos referentes à primeira aula de laboratório. O objetivo aqui foi criar o primeiro programa em Java com interação do usuário.

## 🛠️ O que foi praticado
* **Classe `Scanner`**: Utilizada para ler entradas de texto (`nextLine()`) e números (`nextInt()`) via terminal.
* **Criação de Objetos**: Instanciação da classe `OlaAluno` dentro do método `main` no arquivo `App.java`.
* **Métodos**: Separação de responsabilidades criando métodos distintos (`apresentacaoAluno()` e `dadosAluno()`).

## ▶️ Como rodar apenas este roteiro
Estando na pasta `Roteiro01/src`, compile e execute o arquivo principal:
`javac App.java OlaAluno.java`
`java App`
